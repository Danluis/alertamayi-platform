# 🚀 Alertamed

Alertamed is a healthcare-focused platform designed to improve data management and accessibility in medical environments.

## 🧠 Overview

This project focuses on building a scalable and maintainable system to handle critical medical data and workflows. It is designed with a fullstack architecture, ensuring performance and flexibility for future growth.

## ⚙️ Tech Stack

**Frontend:**
- React
- TypeScript

**Backend:**
- Node.js (Express)

**Database:**
- MySQL

**Infrastructure:**
- Nginx
- DigitalOcean

## 🏗️ Architecture

The system is designed using a modular architecture:

- Separation between frontend and backend layers  
- RESTful API communication  
- Structured database relationships for scalability  
- Focus on maintainability and performance  

## 🔥 Key Features

- User and data management system  
- Secure handling of sensitive data  
- Scalable API design  
- Responsive and user-friendly UI  

## 🚀 My Role

- Designed the fullstack architecture from scratch  
- Developed backend APIs and database structure  
- Built frontend interfaces focused on usability  
- Ensured scalability and performance of the system
